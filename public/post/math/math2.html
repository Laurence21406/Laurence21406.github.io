多项式因式分解

公式法：
$$ (a \pm b)^{2} = a^{2} \pm 2ab +b^{2} $$ $$ (a \pm b)^{3} = a^{3} \pm 3a^{2}b + 3ab^{2} \pm b^{3} $$ $$ a^{2} - b^{2} = (a + b)(a - b) $$ $$ a^{3} \pm b^{3} = (a \pm b)(a^{2} \mp ab + b^{2}) $$ $$ a^{3} + b^{3} + c^{3} - 3abc = (a + b + c)(a^{2} + b^{2} + c^{2} - ab - bc - ac) $$

\begin{flalign} \frac{1}{(x + a)(x + b)} &= \frac{A}{x + a} + \frac{B}{x + b} \\ \frac{1}{(x + a)(x + b)(x + c)} &= \frac{A}{x + a} + \frac{B}{x + b} + \frac{C}{x + c} \\ \frac{1}{(x + a)(x + b)^{2}} &= \frac{A}{x + a} + \frac{B}{x + b} + \frac{C}{(x + b)^{2}} \\ \frac{1}{(x + a)(x^{2} + bx + c)} &= \frac{A}{x + a} + \frac{Bx + C}{x^{2} + bx +c} \\ \frac{1}{(x^{2} + ax + b)(x^{2} + cx + d)} &= \frac{Ax + B}{x^{2} + ax + b} + \frac{Cx + D}{x^{2} + cx + d} \\ \frac{1}{(x + a)(x^{2} + bx + c)^{2}} &= \frac{A}{x + a} + \frac{Bx + C}{x^2 + bx +c} + \frac{Dx + E}{(x^2 + bx +c)^{2}} \\ \frac{1}{(x + a)^{2}(x^{2} + bx + c)^{2}} &= \frac{A}{x + a} +\frac{B}{(x + a)^{2}} + \frac{Cx + D}{x^2 + bx +c} + \frac{Ex + F}{(x^2 + bx +c)^{2}} \\ \end{flalign} 【数学】有理分式的拆解技巧
