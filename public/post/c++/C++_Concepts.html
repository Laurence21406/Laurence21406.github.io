抽象类(Abstract Base Class)

抽象类(Abstract Base Class)： 含有一个或多个纯虚函数/抽象方法, 可以有状态，可以有被实现的方法，不能被实例化
虚函数(virtual function): 虚方法
纯虚函数(pure virtual function): 抽象方法, 子类必须实现, 行为规范 类纯虚函数的声明就是在告诉子类的设计者，"你必须提供一个纯虚函数的实现，但我不知道你会怎样实现它" 。

接口(Interface)

接口(Interface): 只包含虚析构函数(virtual destructor)和纯虚函数/抽象方法(pure virtual function)；没有状态；继承接口必须实现接口所有方法;

抽象类是类的抽象,模板设计

Prefer providing abstract interfaces.
abstract interface: (pure virtual function)

接口是行为抽象(规范); 抽象类是类的抽象,模板设计
