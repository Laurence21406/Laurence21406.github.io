GDB调试入门

启动调试

  1. 启动调试并传入参数：
    • gdb --args < exe > arg1 arg2 "arg3 agr4"
    • set args < args >
    • run < args >
  2. 附加到进程：
    适用于程序已经启动了
    • gdb attach < pid >
    • gdb --pid < pid >
  3. 逐过程执行：
    单步执行(step-over),遇到函数跳过函数
    • next
    • n
  4. 逐语句执行：
    单步执行(step-into),遇到函数进入函数
    • step
    • s
  5. 退出当前函数：
    • finish
  6. 退出调试：
    • detach(分离)
    • quit(退出)
    • q
